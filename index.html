Objective:

Working with Arrays of Objects and Rendering Information on HTML

Task 1: Music Bands

Create an array named bands that contains objects representing different music bands.

Each band object should have the following properties:

  • name (string): The name of the band.
  • genre (string): The genre of music the band plays.
  • formedYear (number): The year the band was formed.
  • active (boolean): Whether the band is currently active.

Render the Information on HTML:

  • Write a function that takes the bands array and renders the information on an HTML page.
  • Display each band's name, genre, and the year they were formed.
  • Indicate whether the band is currently active.

Task 2: Restaurants

Create an array named restaurants that contains objects representing different restaurants.

Each restaurant object should have the following properties:

  • name (string): The name of the restaurant.
  • cuisine (string): The type of cuisine the restaurant serves.
  • location (string): The location of the restaurant.
  • rating (number): The average rating of the restaurant (out of 5).

Render the Information on HTML:

  • Write a function that takes the restaurants array and renders the information on an HTML page.
  • Display each restaurant's name, cuisine, location, and rating.

Task 3: Supermarket Products

Create an array named products that contains objects representing basic supermarket products.

Each product object should have the following properties:

  • name (string): The name of the product.
  • category (string): The category the product belongs to (e.g., dairy, produce, etc.).
  • price (number): The price of the product.
  • inStock (boolean): Whether the product is currently in stock.

Render the Information on HTML:

  • Write a function that takes the products array and renders the information on an HTML page.
  • Display each product's name, category, price, and stock status.

Submission

  • Create an HTML file for each task (bands.html, restaurants.html, products.html).
  • Use a navigation bar
  • Use Bootstrap 5
  • Ensure your HTML and JavaScript files are well-organized and commented.
